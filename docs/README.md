# 文档索引

本目录只保留当前仍可用于开发、验证和发布的文档。事实优先级如下：

1. 运行代码、数据库 schema 与自动化测试；
2. `openapi-final.yaml` 接口快照；
3. 本目录中的说明文档；
4. 历史评估产物仅可作背景，不作为当前结论。

| 文档 | 用途 |
|---|---|
| `current-status.md` | 当前进度、验证结果、风险与声明边界 |
| `architecture.md` | 运行流、目录职责、模块依赖与拆分规则 |
| `api.md` | API 前缀、分组、响应约定与兼容规则 |
| `algorithms.md` | 两套策略、主力控盘度公式、字段与版本约束 |
| `development.md` | 环境、测试、构建、数据库与发布流程 |

维护规则：功能合入时更新对应主题文档，不再在仓库根目录新增一次性“说明书”或“验证报告”。自动生成的 WFF/测试证据放在 `tmp/local-artifacts/`，不与长期维护文档混放。
